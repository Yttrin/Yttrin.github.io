Argvchs的小窝有完整的博客搭建教学，不过我把要做的事情都搞定了才看到…

我主要参照此文完成了这里的搭建。不过，其中遇到了一些问题，我在此进行补充：

  1. 在执行 hexo 时，可能会被 PowerShell 拦截，请使用 Win+R 打开 cmd，再在目录内执行hexo命令。
  2. install hexo-deployer-git 时，可能会出现错误，这时请使用yarn add hexo-deployer-git进行安装。
  3. hexo deploy时，可能无法访问 _config.yml 文件，这时请确认：
  • 本地的 _config.yml 文件中的 deploy 项是否正确配置。例如我的是：
# Deployment
## Docs: https://hexo.io/docs/one-command-deployment
deploy:
  type: 'git'
  repo: git@github.com:Yttrin/Yttrin.github.io.git
  branch: main
  • 确认这个 git 仓库可以写入。例如，我可以在另一个文件夹执行git clone git@github.com:Yttrin/Yttrin.github.io.git以克隆我的仓库。
